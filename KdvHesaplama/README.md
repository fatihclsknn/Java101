# Kdv Tutarı Hesaplayan Program

Java ile kullanıcidan alıinan para degerinin KDV'li
fiyatinı ve KDV tutarını hesaplayip ekrana bastıran
programi yazın.
(Not: tutar <1000 iseKDV tutarinıi 18% olarak alin tutar > 1000 ise %8)
KDV'siz Fiyat = 10;
KDV'li Fiyat = 11.8;
KDV tutar = 1.8;
